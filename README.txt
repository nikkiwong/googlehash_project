To run the program:

Open the folder containing all the files into PyCharm/Eclipse or whatever IDE you wish to use. 

Run the file called sample.py.

To change the file input, go to line 11 and change the data file "me_at_the_zoo.in" to the desired file.

The score given is a best overall score in milliseconds. 

optional:
If you wish you can run the file that computes a number of generations: maximising_score_generation.py.
